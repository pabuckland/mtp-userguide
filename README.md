# MTP User Guide

Welcome to the **Moment Tensor Potentials (MTP) User Guide**.

This repository provides a step-by-step guide for researchers learning to use **Moment Tensor Potentials (MTPs)** on the **Digital Research Alliance of Canada (DRAC)** high-performance computing (HPC) systems.

The guide is designed to bridge the gap between setting up your computer and successfully completing the official MLIP tutorials.

---

## Learning Path

2. Prerequisites

5. Linux Basics

9. MLIP-3 Overview
10. LAMMPS Integration
11. Troubleshooting

## Intended Audience

This guide is intended for:

- Undergraduate researchers
- Graduate students
- New users of DRAC clusters
- Anyone beginning work with MTPs

No previous experience with Linux or HPC systems is assumed.
