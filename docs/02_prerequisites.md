# Prerequisites

Before beginning this guide, you should have the following:

## Required Accounts

- Digital Research Alliance of Canada (DRAC) account
- Access to the Trillium cluster (or another Alliance cluster)
- GitHub account

## Required Software

- Visual Studio Code
- Remote SSH extension
- Windows PowerShell (Windows users)

## Recommended Knowledge

No previous experience with Linux, HPC systems, or Moment Tensor Potentials is required. These topics will be introduced throughout this guide.

The next tutorial will walk through installing and configuring Visual Studio Code.
