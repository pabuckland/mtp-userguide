# Introduction

Welcome to the MTP User Guide.

This repository provides onboarding documentation for researchers learning to use Moment Tensor Potentials (MTPs), MLIP, LAMMPS, and HPC resources through the Digital Research Alliance of Canada (DRAC).

## Purpose

This guide is designed to help new users transition from basic computer setup to running MTP-based simulations.

The learning pathway includes:

- Installing required software
- Connecting to DRAC clusters
- Learning Linux and HPC workflows
- Understanding MLIP-2 and MLIP-3
- Running simulations using LAMMPS
